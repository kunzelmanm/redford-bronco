We went to just look, I swear, and ended up buying. I'd been wanting the Heritage Edition since I saw one, but a deal was to be had for a better equipped Badlands. Rather than paying MSRP and waiting, this was better-equipped and available.
## The Build

**2025 Ford Bronco Badlands Sasquatch**

The Sasquatch package was non-negotiable. 35-inch Goodyear Territory MT tires on beadlock-capable wheels, 4.70 gears, electronic locking front and rear differentials. This thing came with more stuff than we need for what we're planning.

But we didn't stop there.

### HOSS 3.0 Suspension

It already had this installed. It feels like it rides pretty well on the road, we'll see what happens in a little rougher terrain soon.
### Raptor Running Boards

This thing sits pretty high, so April needs steps. This one came with the Raptor style board already installed.

### The Color

Ruby Red Metallic with the color-matched hardtop. Not my first or even second choice. Maybe a white vinyl wrap is in the future.

## What's Next

Ordered some IAG mudflaps to guard against some of the rock chips for a little while. Some paint protection film on a few things to help a little more. It's already got lockers, so maybe some lights!
